\doxysubsection{Detailed Description}
Word dictionary for language models. 

This class has two main purposes\+: (i) store a list of \char`\"{}known\char`\"{} words to be used within a language model and (ii) provide conversions between word and k-\/gram tokens and word and k-\/gram codes (strings of integers), where the latters are employed in the internal implementation of \mbox{\hyperlink{classkgramFreqs}{kgram\+Freqs}} class.
